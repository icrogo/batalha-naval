# Jogo de Batalha Naval

Projeto desenvolvido para a disciplina de Programação.

## Descrição
Jogo simples de Batalha Naval desenvolvido em HTML, CSS e JavaScript.

## Funcionalidades
- Tabuleiro 8x8
- Navios posicionados aleatoriamente
- Sistema de ataque por clique
- Indicação de acerto e erro
- Reinício de jogo
